## 🏗️ 技术架构亮点

### 五层分层架构
```
接入层 (Access Layer)
    ↓
应用层 (Application Layer)
    ↓
业务层 (Business Layer)
    ↓
平台层 (Platform Layer)
    ↓
基础层 (Infrastructure Layer)
```

### 核心技术栈
- **前端**: React 18 + TypeScript + Ant Design
- **后端**: Python (FastAPI) + Go (Gin/gRPC)
- **AI/ML**: vLLM + Transformers + PyTorch
- **数据**: PostgreSQL + Redis + Milvus + Kafka
- **基础设施**: Kubernetes + Istio + Prometheus

### 7大核心组件
1. **智能路由引擎** - Query分类 + 多模型协同编排
2. **成本管控系统** - Token级计量 + 智能降本
3. **安全防护体系** - 多层防护 + 红蓝对抗
4. **AIOps运维系统** - 异常检测 + 自愈机制
5. **评测体系** - 自动化评测 + 多维指标
6. **开发者工具** - 可视化编辑 + 时光机调试
7. **RAG引擎** - 混合检索 + 知识图谱增强
