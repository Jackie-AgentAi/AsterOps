## 🏗️ 技术架构

### 前端技术栈
```
Vue 3.4 + TypeScript + Vite
├── UI框架: Element Plus
├── 状态管理: Pinia
├── 路由: Vue Router 4
├── HTTP: Axios + 拦截器
├── 图表: ECharts + Vue-ECharts
├── 缓存: 多级缓存系统
├── WebSocket: 实时通信
└── 监控: 健康检查组件
```

### 后端集成架构
```
API网关 (Kong + Go)
├── 路由转发: /v1-6/* → 微服务
├── 认证授权: JWT Token
├── 限流熔断: 请求保护
└── 监控日志: 请求追踪

微服务集群
├── 用户服务 (v1): 8081
├── 项目服务 (v6): 8082  
├── 模型服务 (v2): 8083
├── 推理服务 (v3): 8084
├── 成本服务 (v4): 8085
└── 监控服务 (v5): 8086
```

## 🚀 核心功能亮点

### 1. **智能缓存系统**
- **多级缓存**: 内存 → LocalStorage → SessionStorage
- **智能失效**: 基于操作类型的自动缓存清理
- **性能优化**: LRU算法和TTL管理
- **统计监控**: 缓存命中率和性能指标

### 2. **健壮的错误处理**
- **自动重试**: 指数退避算法，最多3次重试
- **错误分类**: 网络错误、服务器错误、业务错误
- **降级处理**: 失败时返回缓存数据
- **用户友好**: 清晰的错误提示和恢复建议

### 3. **实时通信系统**
- **WebSocket管理**: 自动连接、重连、心跳检测
- **事件订阅**: 系统状态、告警、推理结果等
- **状态同步**: 实时更新服务状态和用户界面

### 4. **健康监控系统**
- **服务发现**: 自动检测微服务状态
- **性能监控**: 响应时间、成功率、资源使用
- **告警机制**: 服务异常时自动告警
- **可视化**: 实时状态显示和详细监控面板

## 🚀 部署和运行

### 开发环境
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 访问地址
http://localhost:3000
```

### 生产环境
```bash
# 构建项目
npm run build

# 使用Docker部署
docker-compose -f docker-compose.frontend.yml up -d

# 访问地址
http://localhost:3000
```

### 后端集成测试
```bash
# 启动完整集成环境
./scripts/start-frontend-integration.sh

# 执行集成测试
./scripts/test-frontend-integration.sh
```
