### 3. 菜单结构优化

#### 3.1 用户管理主菜单
```
用户管理 (主菜单)
├── 用户列表 (子菜单)
│   ├── 用户管理 - 用户CRUD操作、搜索、筛选
│   ├── 用户详情 - 用户信息查看、编辑
│   ├── 用户导入导出 - 批量用户操作
│   └── 批量操作 - 批量状态修改、权限分配
└── 用户组 (子菜单)
    ├── 用户组管理 - 用户组CRUD操作
    ├── 用户组权限 - 组权限分配和管理
    ├── 用户组成员 - 组成员管理
    └── 用户组分析 - 组使用统计和分析
```

#### 3.2 设计原则
- 功能分离：将用户管理和用户组管理分离
- 逻辑清晰：每个子菜单职责明确
- 操作便捷：相关功能集中，减少页面跳转
- 扩展性好：为未来功能扩展预留空间

## API接口列表

### 用户组管理
- `GET /api/v1/user-groups` - 获取用户组列表
- `POST /api/v1/user-groups` - 创建用户组
- `GET /api/v1/user-groups/:id` - 获取用户组详情
- `PUT /api/v1/user-groups/:id` - 更新用户组
- `DELETE /api/v1/user-groups/:id` - 删除用户组
- `GET /api/v1/user-groups/search` - 搜索用户组

### 用户组成员管理
- `GET /api/v1/user-groups/:id/members` - 获取用户组成员
- `POST /api/v1/user-groups/:id/members` - 添加用户组成员
- `DELETE /api/v1/user-groups/:id/members/:user_id` - 移除用户组成员
- `GET /api/v1/user-groups/user/:user_id` - 获取用户所属的用户组
